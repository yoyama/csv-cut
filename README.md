# csv-cut

# csv convert tool

```
  -i <value> | --input <value>
     input file name
  -l <value> | --line <value>
     line for horizontal split
  -f <value> | --field <value>
     output fields. output fields. (ex1) -f 1,2,3 (ex2) -f 2-5 (ex3) -f 3-
  --no-header-in
     no header for input
  --no-header-out
     no header for output
  --prefix-out <value>
     prefix for output file name. default 'out'
  --suffix-out <value>
     suffix for output file name. default '.csv'
  --enc-out <value>
     output encode (utf | sjis | euc)
  --crlf-out <value>
     output crlf  (unix | win | mac)
  --help
     prints this usage text
```
